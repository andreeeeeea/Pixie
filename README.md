# Pixie

A Python-based AI agent that controls your computer using natural language commands. Built with Google Gemini API and PyAutoGUI, Pixie can open applications, type text, click UI elements, take and analyse screenshots, all through conversational interactions.

**Features:**

- Natural language computer control
- Windows application launching via registry lookup
- Keyboard/mouse automation (clicking, typing, key presses)
- Screenshot capture for visual analysis
- Agent loop with multi-step reasoning

**Tech Stack:** Python, Google Gemini 2.5 Flash, PyAutoGUI, Tesseract OCR

### Planned:

- Voice commands
- OCR text finding
- Web browser search
- Vision-based clicking (foundation already in code)

This is currently a work in progress project, but the mentioned features are already implemented. Compatible only with Windows as of now.
